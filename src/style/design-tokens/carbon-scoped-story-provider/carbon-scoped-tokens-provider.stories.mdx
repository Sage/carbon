# Carbon Scoped Tokens Provider

This is temporary compatibility solution that provides Theme properties mapped to Design Tokens in form of CSS Variables in scope wrapped by the provider. This tokens provider requires `CarbonProvider` to provide theme.

## Contents

- [Quick Start](#quick-start)
- [Examples](#examples)

## Quick start

```javascript
import CarbonProvider from "carbon-react/lib/components/carbon-provider";
import CarbonScopedTokensProvider from "carbon-react/lib/style/design-tokens/carbon-scoped-tokens-provider";
```
