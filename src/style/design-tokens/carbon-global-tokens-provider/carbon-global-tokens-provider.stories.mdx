# Carbon Global Tokens Provider

This is temporary compatibility solution that provides Theme properties mapped to Design Tokens in form of CSS Variables in global `:root {}` scope. This tokens provider requires `CarbonProvider` to provide theme.

If you want to narrow scope of tokens, you use <LinkTo kind="Documentation/Token Providers/Carbon Scoped Tokens Provider">CarbonScopedTokensProvider</LinkTo>.

## Contents

- [Quick Start](#quick-start)
- [Examples](#examples)

## Quick start

```javascript
import CarbonProvider from "carbon-react/lib/components/carbon-provider";
import CarbonGlobalTokensProvider from "carbon-react/lib/style/design-tokens/carbon-global-tokens-provider";
```
