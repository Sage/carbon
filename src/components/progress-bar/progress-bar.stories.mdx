# ProgressBar

Use the ProgressBar component to let users know a task or loading data is still in progress.
Showing a ProgressBar helps the user to understand that they should wait, rather than reload the page or abandon a process.
In general, place a ProgressBar in the centre and middle of the page or container it relates to.

## Quick Start

Import `ProgressBar` into the project.

```javascript
import ProgressBar from "carbon-react/lib/components/progress-bar";
```
