## Examples

### Default

To use this component, pass `onPagination`, `totalRecords`, and `currentPage`.
The `onPagination` callback is fired whenever navigation occurs and provides the selected `currentPage`, the `pageSize` (records per page), and the `origin` of the interaction.

The total number of pages is calculated from the total number of records and the selected page size.

<Canvas of={PagerStories.Default}/>

### With Page Size Selection

Use `pageSize` prop to define how many records are shown per page.

To allow users change this instead, set `showPageSizeSelection` to `true` to render a dropdown with a set of page size options.

To set your own custom set of page size options, pass them in an array to `pageSizeSelectionOptions`.

<Canvas of={PagerStories.WithPageSizeSelection}/>

### Non-Interactive Page

To disable direct editing of the current page number, you can set `interactivePageNumber` to `false`.

<Canvas of={PagerStories.NonInteractivePage}/>

### Hide First and Last Buttons

To remove the First and Last page navigation buttons, set the `showFirstAndLastButtons` to `false`.

<Canvas of={PagerStories.HideFirstAndLastButtons}/>

### Alternate Variant

The `"alternate"` variant can be set to render the component with a transparent border and background. 

<Canvas of={PagerStories.AlternateVariant}/>

Use the `size` prop to change the size of the component, available options are `"small"`, `"medium"` (default), and `"large"`.
