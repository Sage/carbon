# Row & Column

Useful to organise the UI of a page into a simple column-based layout.

Configure the number of columns, the margin between them, and toggle separators visibility.

## Contents

- [Quick Start](#quick-start)
- [Examples](#examples)
- [Props](#props)

## Quick Start

```javascript
import { Row, Column } from "carbon-react/lib/components/row";
```

### With offset columns

To offset a column besides passing a `columnOffset` prop on wanted `Column` it is also necessary to provide `columns` prop to `Row` component with a number of resulting columns.
