# Alert

<DeprecationWarning>
Alert has been deprecated, if this pattern is still needed please see <a style={{color: "white"}} href="../?path=/docs/documentation-deprecation-migration--docs#alert">our deprecation migration docs</a> for an alternative pattern.
</DeprecationWarning>

A deprecated small modal dialog that displays a message with a title, subtitle and a close button. Use `Confirm` or `Dialog` for new implementations.

**Category:** Modal
