# Tile

Tiles are containers for static content such as text, and don't contain controls.

## Contents

- [Quick Start](#quick-start)
- [Examples](#examples)
- [Props](#props)

## Quick Start

```javascript
import Tile from "carbon-react/lib/components/tile";
import TileFooter from "carbon-react/lib/components/tile/tile-footer";
```

## Designer notes

- Tiles should be reserved for static content. If you require a container with controls or interactive content, use the <LinkTo kind='Card' story='default-story'>Card component</LinkTo> instead.

### With custom widths

The Tile component can also have a custom `width` which accepts either numbers or strings: any decimal between 0 and 1.0
will be converted into a percentage, any number grater will be be treated as pixels and any valid CSS string will also work.
Any child wrapped by a Tile component can be passed an optional `width` prop.
Please note that it will not have any effect if the Tile orientation is set to `vertical`.

<Canvas>
