A Button triggers a single action or event.
Use it to submit a form (Save), to advance to the next step in a process (Next), or to create a new item (New).

**This documentation is for the newer implementation of Button. If you are still using the older implementation, please use the <a href="../?path=/docs/deprecated-button--docs">Button documentation</a>.**
