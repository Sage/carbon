## Examples

### Default

The `email` prop automatically surfaces the `href` attribute with `mailto:`. This will enable
users to send a new email within their default email program to the email address. To display an
email address without `mailto:`, use custom content instead of providing the `email` prop.

<Canvas of={ProfileStories.Default} />

### Variant

The `variant` prop can be passed to apply a colour variant to the Portrait.
See the [Portrait](/docs/portrait--docs) docs for more information on variant.

<Canvas of={ProfileStories.Variant} />

### Src

To use an image, simply pass any valid image URL as a `src` prop.
