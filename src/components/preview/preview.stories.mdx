# Preview

Applies a preview loading state animation.

## Quick Start

Import `Preview` into the project.

```jsx
import Preview from "carbon-react/lib/components/preview";
```

### Default

<Preview>
  <Story name="default">
    <PreviewComponent loading />
  </Story>
</Preview>

### Preview with Lines

<Preview>
  <Story name="with Lines">
    <PreviewComponent loading lines={6} />
  </Story>
</Preview>

### Preview with Children

<Preview>
  <Story name="with Children">
    <PreviewComponent>
      This the where the children are rendered
    </PreviewComponent>
  </Story>
</Preview>

  <Story name="with Width">
