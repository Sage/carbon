# Breadcrumbs

Breadcrumbs are a secondary navigation aid that helps users easily understand their location.

## Contents

- [Quick Start](#quick-start)
- [Examples](#examples)
- [Props](#props)

## Quick Start

To use Breadcrumbs component you need to import `Breadcrumbs` and use `Link` component as a child.

```javascript
import Breadcrumbs from "carbon-react/lib/components/breadcrumbs";
```
