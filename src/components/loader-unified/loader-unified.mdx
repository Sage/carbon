# Unified Loader
Use the `UnifiedLoader` component to clearly indicate that a task or data is still loading, helping users understand they should wait rather than refresh the page or abandon the process. The `UnifiedLoader` component offers three loading types: `standalone` (with `typical` and `ai` variants), `ring` (with `inline` or `stacked` variants) and `star` for clear visual feedback.

## Quick Start

Import `UnifiedLoader` into the project.

```javascript
import UnifiedLoader from "carbon-react/lib/components/loader-unified";
```

### Loader Sizes

The UnifiedLoader component offers size flexibility based on the selected loader type: the Standalone loader supports three sizes—Small, Medium, and Large—while the Ring loader provides four options—Extra Small, Small, Medium, and Large. The Star loader type has a single default size, ensuring a consistent appearance across use cases.
