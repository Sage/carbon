CheckboxGroup provides a way to check one or more items from a list of options.

## Contents

## Quick Start

```javascript
import { Checkbox, CheckboxGroup } from "carbon-react/lib/components/checkbox";
```

## Examples

### Default

To use this component, pass the `Checkbox` inputs as children of `CheckboxGroup`.

<Canvas of={CheckboxGroupStories.Default} />

### With Legend

A legend can be set on the group by passing the `legend` prop to `CheckboxGroup`.

<Canvas of={CheckboxGroupStories.WithLegend} />

### With Legend Hint

To add a hint below the legend, pass the `legendHint` prop to `CheckboxGroup`.

<Canvas of={CheckboxGroupStories.WithLegendHint} />

### Inline

`Checkbox` children can be displayed inline by passing the `inline` prop to `CheckboxGroup`.

<Canvas of={CheckboxGroupStories.Inline} />

### Sizes

The size of the component can be changed by passing the `size` prop to `CheckboxGroup`. Available options are `small`, `medium`, and `large`.
Note that any size set on individual `Checkbox` children will be overridden.

<Canvas of={CheckboxGroupStories.Sizes} />

### Required

`CheckboxGroup` can be marked as required by passing the `required` prop.

<Canvas of={CheckboxGroupStories.Required} />


### Disabled

`CheckboxGroup` can be disabled using the `disabled` prop.

**Please Note**: Even though Carbon does support disabled checkboxes, their use is not generally recommended by Design System.


<Canvas of={CheckboxGroupStories.Disabled} />
