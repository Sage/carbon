# RadioTile
Is a deselectable radio button input visualized as a single or grouped set of tiles

## Quick Start

To use this component, import the `RadioTile` and `RadioTileGroup` if you want to have `RadioTiles` grouped.

```jsx
import { RadioTileGroup, RadioTile } from "carbon-react/lib/components/radio-tile"

const MyComponent = () => (
  const [value, setValue] = useState(null);
  <RadioTileGroup
    name="MyInputName"
    value={value}
    legend="Radio Tile"
    description="Pick one of the available options"
    onChange={(e) => setValue(e.target.value)}
  >
    <RadioTile 
      value="1"
      title="Title"
      subtitle="Subtitle"
      description="Short and descriptive description"
    />
    <RadioTile 
      value="2"
      title="Title"
      subtitle="Subtitle"
      titleAdornment={<Pill>Message</Pill>}
      description="Short and descriptive description"
    />
  </RadioTileGroup>
);
```

## Examples

### Single Select

In single select mode all input props like `onChange`, `onBlur`, `name` and currently selected `value` are meant to be passed on the `RadioTileGroup`.

These props are internally distributed on each of the `RadioTiles` making whole composed group act like a single form field.

### Multi select
To enable multi select mode `multiSelect` boolean prop has to be passed as `true`.

Also all input props like `onChange`, `onBlur`, `name`, `value` and `checked` are meant to be passed individually on each of the `RadioTiles` instead of on the `RadioTileGroup`.

In this mode `RadioTileGroup` serves only a visual purpose - it only renders `legend` and `description` props and applies spacing to each of the `RadioTiles`

### Single tile
`RadioTile` can be also used as a single component

Same as in multi select example grouped by `RadioTileGroup`, `onChange`, `onBlur`, `name`, `value` and `checked` are meant to be passed directly on the `RadioTile`.

<Preview>
  <Story name="Single tile">
    {() => {
      const [isChecked, setIsChecked] = useState(false);
      return (
        <RadioTile 
          value="1"
          name="single"
          title="Title"
          subtitle="Subtitle"
          description="Short and descriptive description"
          checked={isChecked}
          onChange={e => setIsChecked(e.target.checked)}
        />
      )
    }}
  </Story>
</Preview>
