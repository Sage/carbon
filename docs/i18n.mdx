# i18n

## Default

By default `carbon` uses `en-GB` translations.

Preview [src/locales/en-gb.js](https://github.com/Sage/carbon/blob/master/src/locales/en-gb.js)

## Resources

To add a new locale and use it with `carbon`, create a new locale file based on [interface](https://github.com/Sage/carbon/blob/master/src/components/i18n-provider/i18n-provider.d.ts) or [proptypes](https://github.com/Sage/carbon/blob/master/src/components/i18n-provider/i18n-provider.component.js) of `I18nProvider` component.
The default locale file `en-GB` will be **extended** with the new locale file.

## Usage

```jsx
import I18nProvider from "carbon-react/lib/components/i18n-provider";

import enUS from "./locales/en-us";
import App from ".";

<I18nProvider locale={enUS}>
  <App />
</I18nProvider>;
```

or

```jsx
import I18nProvider from "carbon-react/lib/components/i18n-provider";
import Switch from "carbon-react/lib/components/switch";

/* ... */

<I18nProvider
  locale={{
    locale: () => "fr-FR",
    switch: {
      on: () => "sur",
      off: () => "de",
    },
  }}
>
  <Switch onChange={handleChange} name="switch-default" checked={isChecked} />
</I18nProvider>;
```
